A Lotta Keys

Welcome to my site!

My name is Sinan, but you likely know me as Hugged from osu!mania. This is a personal site first and foremost. I want to use it as a platform to host my projects. If you're interested in the lore and design of this site and its backend in detail, check out the About page!

A Lotta Keys is essentially a project to archive the history of osu!mania's 10K+ subcommunity in data. I am making daily requests for recent 10K+ mania scores using the osu! API. Most of these scores get deleted from osu!'s database after 24 hours. I intend to preserve everything I can scrape, and to allow users of this site to view all those cool stats with the power of data visualization.
